## Selected Projects

### Resume Ranking System — Final Year Project

An AI-powered talent matching platform that semantically ranks candidate resumes against job descriptions. Built a hybrid retrieval architecture combining multi-entity vector embeddings, BM25 ranking, reciprocal rank fusion, cross-encoder reranking, and LLM-driven structured extraction via schema-enforced prompting (BAML). The backend is a microservices architecture using Node.js/Express and Python FastAPI, with PostgreSQL, Qdrant vector database, Docker Compose, JWT authentication, and structured logging. Deployed live on AWS EC2.

`Python` `FastAPI` `Node.js` `Qdrant` `LLMs` `RAG` `Docker` `PostgreSQL`

### MLOps AQI Prediction System
[github.com/AbdullahIqbal26904/MLOPS_Project](https://github.com/AbdullahIqbal26904/MLOPS_Project) · **Live:** [56.228.3.203:3000](http://56.228.3.203:3000)

A production-grade MLOps and LLMOps pipeline for real-time Air Quality Index forecasting. Ingests hourly weather data from OpenWeatherMap and OpenMeteo APIs, stores features in AWS S3, trains XGBoost models with 72-hour prediction horizons, and serves predictions via FastAPI. The system includes a full observability stack — MLflow for experiment tracking, Evidently AI for data drift monitoring, and Prometheus/Grafana for infrastructure metrics — orchestrated via Docker Compose with CI/CD through GitHub Actions. Enhanced with a RAG chatbot (Llama 3.3 via Groq, ChromaDB) with PII detection and hallucination filtering guardrails. Deployed on AWS EC2 (Ubuntu 22.04, t3.medium).

`Python` `XGBoost` `MLflow` `Evidently AI` `FastAPI` `Docker` `AWS EC2/S3` `Prometheus` `Grafana` `RAG` `ChromaDB`

### AI Maze Solver — Ant Colony Optimisation and A*
[github.com/AbdullahIqbal26904/Maze-Solver-Using-Aco-and-A-](https://github.com/AbdullahIqbal26904/Maze-Solver-Using-Aco-and-A-)

Comparative implementation of two pathfinding strategies — Ant Colony Optimisation (ACO) and A* — applied to maze traversal. The project analyses convergence behaviour, computational complexity, and solution quality of both approaches across maze configurations of varying size and density.

### Huffman Coding File Compression
[github.com/AbdullahIqbal26904/DataStructure-Project-Huffman-coding](https://github.com/AbdullahIqbal26904/DataStructure-Project-Huffman-coding)

Implementation of lossless file compression and decompression using the Huffman coding algorithm. Constructs an optimal prefix-free binary tree from symbol frequency analysis, achieving significant reduction in file size while maintaining full data integrity on reconstruction.
